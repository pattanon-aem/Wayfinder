# Wayfinder – Smart Transport Navigation Platform
[🌐 Live Demo](https://www.wayfinder.ink)
---

## Overview

Wayfinder is a web-based smart transport navigation platform that provides **real-time, context-aware route recommendations** for commuters in Singapore.

Unlike traditional navigation systems, Wayfinder integrates multiple live data sources and AI-based decision-making to help users choose the most efficient travel option across different transport modes.

---

## Features

### Real-Time Location & Map

* Detects user location using browser geolocation
* Interactive map interface powered by Google Maps
* Displays nearby transport options dynamically

---

### Live Transport Data

* Real-time bus arrival timings
* Taxi stand availability
* Carpark availability tracking

---

### Smart Search & Navigation

* Search destinations with auto-suggestions
* Multi-modal route planning:

  * Public Transport
  * Driving
  * Walking
  * Cycling

---

### AI-Powered Recommendations

* Suggests optimal travel mode based on:

  * Travel time
  * Cost
  * Traffic conditions
* Provides explanation for recommendations

---

### User Account System

* Secure authentication (email & Google OAuth)
* Manage profile and saved addresses
* Personalized travel experience

---

## System Architecture

Wayfinder integrates multiple external services:

* **Google Maps API** – Map visualization & routing
* **LTA DataMall API** – Real-time transport data
* **MongoDB Atlas** – Database storage
* **Clerk Authentication** – User management
* **AI Service API** – Travel recommendation engine
